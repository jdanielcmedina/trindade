# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-01

### Added
- Initial release of Trindade Framework
- Core system with routing and middleware support
- Database abstraction using Medoo
- Session and Cookie management
- Cache system with File, Redis and Memcached drivers
- Mail system using PHPMailer
- Logging system with multiple levels
- Asset management
- Internationalization support
- File upload handling
- Utility functions
- Blog plugin as example
- Complete documentation
- Unit tests

### Security
- CSRF protection
- XSS prevention
- SQL injection protection via Medoo
- Secure session handling
- Password hashing with bcrypt
